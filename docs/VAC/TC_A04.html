Test Case A 04: HCP validates the vaccination (TBE) (Doc6) which was entered by the patient (Doc5)

Precondition

In the vaccination documentation of the patient, which is stored in the ehealth ecosystem, there is also the document with a TBE vaccination given by a doctor and entered by the patient. This entry must be validated by a health professional. The documents of the Test Cases A01, A02, A03 are prerequisite.

Sequence

The healthcare professional creates a new Document A-D6-HCP1-C1 which refers to the Document A-D5-P-C1 created by the patient (Compositions.relatesTo), as well as using the extension Extension ch-core-ext-entry-resource-cross-references to set the two immunization entries in relation to one another and the date Immunization.recorded are set as new. The document is stored in the ehealth ecosystem.

Result

The 2nd dose of the TBE vaccination has now been validated by a health professional.
