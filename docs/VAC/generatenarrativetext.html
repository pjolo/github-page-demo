Generatenarrativetext

Narrative Text Generation

XML to HTML Transform

To transform FHIR XML CH VACD document to html you can use ChVacdDocumentToHTML XSL transformer.

use i.e. the following command to do so:

xsltproc ChVacdDocumentToHTML.xslt -o html-output-file fhir-xml-input-file

To generate the narrative content from a document, you can process the following XSL transformer ChVacdDocumentToCompositionNarrative.

xsltproc ChVacdDocumentToCompositionNarrative.xslt -o html-output-file fhir-xml-input-file

The output can be added as content (stripe html and body tag) to the narrative text div of the composition in the document bundle.

Examples with the generated narrative can be seen here: Immunization Administration Document and Vaccination Record
